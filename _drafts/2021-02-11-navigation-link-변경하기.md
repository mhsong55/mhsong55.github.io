# Masthead 리스트 파일

`_data/navigation.yml` 파일이 Masthead의 내용을 정의한다.

```yaml
main:
    - title: "Quick-Start Guide"
      url: /docs/quick-start-guide/
    - title: "Posts"
      url: /year-archive/
    - title: "Categories"
      url: /categories/
    - title: "Tags"
      url: /tags/
```
